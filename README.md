## 🎮 **How to Use**

### 1. **Create Your Diagram**
- Type Mermaid syntax in the editor (left panel)
- Use **auto-suggestions** (type 2+ characters to see options)
- Navigate suggestions with ↑↓ arrows, press Enter to apply

### 2. **Customize Appearance**
- Open the **🎨 Diagram Customization** panel
- Switch between **Colors**, **Styling**, and **Layout** tabs
- Adjust settings and see real-time preview
- Save your favorite styles as presets

### 3. **Export & Share**
- **Quick Export**: Use Ctrl+S for instant SVG export
- **Advanced Export**: Click "More Export Options" for PDF, PNG, etc.
- **Share**: Click 🔗 to generate a shareable URL
- **Copy**: Use "Copy to Clipboard" for easy pasting

### 4. **Keyboard Shortcuts**
- **Ctrl+Enter**: Generate preview
- **Ctrl+S**: Quick export SVG
- **Ctrl+Z/Y**: Undo/Redo
- **Ctrl+0**: Reset zoom
- **F11**: Toggle fullscreen
- **⌨️ Button**: View all shortcuts

## 🛠️ **Technical Features**

### 🏗️ **Architecture**
- **Single HTML File**: No build process, runs anywhere
- **Progressive Enhancement**: Graceful degradation if libraries fail
- **Performance Optimized**: Smart rendering for large diagrams
- **Memory Efficient**: Optimized for long editing sessions

### 🔧 **Dependencies**
- **[Mermaid.js v10.8.0](https://mermaid.js.org/)** - Diagram rendering engine
- **[Prism.js](https://prismjs.com/)** - Syntax highlighting
- **[jsPDF](https://github.com/parallax/jsPDF)** - PDF generation
- **[FileSaver.js](https://github.com/eligrey/FileSaver.js/)** - File downloads
- **[Canvg](https://github.com/canvg/canvg)** - SVG to Canvas conversion

### 🔒 **Security & Privacy**
- **No Server Required**: Everything runs in your browser
- **Local Storage Only**: Your data never leaves your device
- **XSS Protection**: Safe URL encoding for sharing
- **Input Sanitization**: Secure handling of user content
